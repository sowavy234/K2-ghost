# K2 SE Automagic Enhancement Suite

**One command. All features. Zero headache.**

- Fully automated, one-run installer for your K2 SE
- No prompts, no questions, **just works!**
- Klipper unlock, DXC-adaptive mesh, flat bed macros, Gumpoy screen install, adaptive fan—fully safe for the K2 SE, and YOUR branding
- Installs custom Fluidd fork and on-printer UI

---

## 🚀 Quick Start

SSH into your K2 SE and run:

```bash
git clone https://github.com/YOUR_USERNAME/K2SE-AutoMagic.git
cd K2SE-AutoMagic
sh install_all_k2se.sh
```

_(Replace `YOUR_USERNAME` with your GitHub name)_

---

## Features

- **Automated Bed Leveling & Flatness:** Insta-mesh with errorproof bed macro
- **DXC Extruder adaptive macros**
