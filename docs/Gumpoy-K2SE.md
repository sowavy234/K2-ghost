# Gumpoy (Gumpy) Screen for K2 SE

- Physically wire and connect as described for K2 SE EXP ports
- UI and branding files in `gumpoy_ui/` auto-deploy with installer
- Configuration and guide provided here and in macros.

**Example macro integration:** Use the AUTO_BED_LEVEL on-screen button to run precise mesh leveling for best results.

**Diagnostic tips:** If the screen is blank, check ribbon cable orientation and power supply.
